JavaScript表达式和运算符

表达式

表达式是一种JS短语，可使JS解释器用来产生一个值。——《JavaScript权威指南》

  • 原始表达式

    • 常量、直接量 345, string
    • 关键字 null，this
    • 变量 a, b, c

  • 复合表达式 # 原始表达式+运算符 a + b

  • 数组、对象的初始化表达式

  • 函数表达式 var foo = function(){};

  • 属性访问表达式 var a = {x: 1}; 可以通过 a.x、a['x'] 访问对象的属性

  • 调用表达式 function();

  • 对象创建表达式 new Func(x, y);

运算符

按照操作数的数量分类：

  • 一元 - x
  • 二元 a + b
  • 三元 c ? a : b

按照运算符的功能分类：

  • 赋值 x += 1 # =、+=、-=
  • 比较 x == y # ==、<、>
  • 算术 x - y # +、-、*、/、%
  • 位 a | b
  • 逻辑 exp1 && exp2
  • 字符串 "a" + "b"
  • 特殊 delete obj.x new in instanceof
